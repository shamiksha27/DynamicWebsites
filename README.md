# Dynamic Web Projects

This repository contains two interactive and dynamic web applications built using modern front-end technologies. These projects focus on user interaction, real-time updates, and a clean UI experience.

---

##  Projects Included

###  1. Digital Clock with Theme Control

A real-time digital clock that updates every second and allows users to switch between different themes.

####  Features:

* Live time display (hours, minutes, seconds)
* Dynamic updates using JavaScript
* Multiple theme modes (e.g., light/dark/custom)
* Smooth and responsive UI
* User-friendly theme toggle controls

####  Technologies Used:

* HTML
* CSS
* JavaScript

---

###  2. Quiz Application

An interactive quiz app that simulates an exam-like environment where users can answer questions and view their final score.

####  Features:

* Multiple-choice questions
* Dynamic question rendering
* Tracks user answers
* Displays final score at the end
* Clean and simple exam-style interface
* Instant feedback after completion

####  Technologies Used:

* HTML
* CSS
* JavaScript

---

## How to Run the Projects

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Open the project folder:

   ```bash
   cd your-repo-name
   ```

3. Open the HTML files in your browser:

   * `digital-clock/index.html`
   * `quiz-app/index.html`
